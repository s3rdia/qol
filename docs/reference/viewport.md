# Set Up Viewports To Draw On

`setup_nested_diagram_viewport()`: Setup the main diagram viewport based
on the list of arguments passed on by
[`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).
This viewport contains the main diagram area as well as the axes.
Additionally sets up the main diagram area viewport which only contains
the main drawing area.

`back_to_the_root()`: Pops out of all viewports.

## Usage

``` r
setup_nested_diagram_viewport(arguments, diagram_info)

back_to_the_root(to_main = TRUE)
```

## Arguments

- arguments:

  Argument list passed on by
  [`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

- diagram_info:

  The list of measurements generated by
  [`get_diagram_dimensions()`](https://s3rdia.github.io/qol/reference/diagram.md).

- to_main:

  TRUE by default. Pops out of all viewports but the main canvas.

## Value

Returns the grid::current.vpPath.
