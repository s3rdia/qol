# Generate The Main Graphical Objects

These functions create the graphical objects for the main diagram area
of the different diagram types. These functions don't work on their own.
They have to be used inside a custom diagram function, which builds the
whole diagram and can be plugged into the "diagram" parameter of
[`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

`vbar_grob()`: Set up the main segments for vertical bars.

## Usage

``` r
vbar_grob(diagram_info, arguments)
```

## Arguments

- diagram_info:

  The list of measurements generated by
  [`get_diagram_dimensions()`](https://s3rdia.github.io/qol/reference/diagram.md).

- arguments:

  Argument list passed passed on by
  [`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

## Value

Returns a grid::gList containing segments and labels.
