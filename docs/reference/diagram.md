# Get The Dimensions And Positions Of Groups And Segments

`get_diagram_dimensions()`: Get various parameters concerning the
division and position of groups and segments. These are e.g. number of
groups/segments, width of groups/segments, ids or positions among other
things.

`setup_interactive_elements()`: Set up additional tooltip zones for
interactive graphics. These additional zones enable a mouse over tooltip
for entire segment groups.

## Usage

``` r
get_diagram_dimensions(arguments)

setup_interactive_elements(arguments, diagram_info)
```

## Arguments

- arguments:

  Argument list passed on by
  [`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

- diagram_info:

  The list of measurements generated by `get_diagram_dimensions()`.

## Value

Returns a list of parameters for groups, segments and axes.

Returns a grid::gList containing tooltip grid::rectGrobs.
