# Generate The Main Diagram Area

The listed functions are the built in diagram types which ship with this
package. These functions don't work on their own and have to be plugged
into the "diagram" parameter of
[`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

`dg_vbars()`: Grouped vertical bars.

## Usage

``` r
dg_vbars(arguments, diagram_info)
```

## Arguments

- arguments:

  Argument list passed on by
  [`design_graphic()`](https://s3rdia.github.io/qol/reference/design_graphic.md).

- diagram_info:

  The list of measurements generated by
  [`get_diagram_dimensions()`](https://s3rdia.github.io/qol/reference/diagram.md).

## Value

Returns a grid::gTree object.

## Examples

``` r
# Example data frame
my_data <- dummy_data(100)

# Formats
age. <- discrete_format(
    "Total"          = 0:100,
    "under 18"       = 0:17,
    "18 to under 65" = 18:64,
    "65 and older"   = 65:100)

sex. <- discrete_format(
    "Total"  = 1:2,
    "Male"   = 1,
    "Female" = 2)

# Design grouped vertical bar chart
qol_graphic <- my_data |>
     design_graphic(axes_variables = "sex",
                    segments       = "age",
                    values         = weight,
                    diagram        = dg_vbars,
                    formats        = list(sex = sex.,
                                          age = age.))
```
